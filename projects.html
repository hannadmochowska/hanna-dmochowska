Closed-Loop Auditory Stimulation (CLAS) — SO Phase Targeting

Goal: deliver tones near a target slow-oscillation phase despite system latency. Streaming “replay” script + synthetic ground truth for phase-error evaluation.

  • Band-pass 0.5–1.5 Hz · Hilbert phase · latency compensation
  • Outputs: phase-error distribution (°), jitter, robustness vs SNR

Aperiodic Spectral Slope Pipeline (REM)

Pipeline for estimating the aperiodic (1/f) slope during REM from hippocampal LFP. Uses Welch power spectra and (optionally) FOOOF to separate periodic peaks from the aperiodic background. Includes scripts for parameter reporting and reproducible fits.

  • Data scope: REM-labeled segments; hippocampal recordings
  • Methods: Welch PSD → aperiodic slope (log–log fit or FOOOF)
  • Analyses: RGS14 control vs positive; OS vs Homecage (exploratory)
  • Findings: trends toward flatter slopes in RGS14-positive and OS, not statistically significant
